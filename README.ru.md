# UTL — Unified Turkic Latin / Единая Тюркская Латиница

**Версия 1.15** | Разработано для оптимальной совместимости с AI-системами (LLM).

---

## История изменений (Changelog)

* **v1.1 – v1.3**: Добавлена буква Jj для /j/ и Žž для /ʒ/. Введена двухуровневая архитектура: UTL-O (орфографический уровень) и UTL-P (фонетический уровень).
* **v1.4 – v1.6**: Замена Әә на Ää для улучшения токенизации. Переход на единую диакритику (гачек) для шипящих: Č, Š, Ž.
* **v1.7**: Решение проблемы «турецкого i»: I всегда соответствует переднему ряду /i/, а Ї — заднему ряду /ɯ/ (ы).
* **v1.8 – v1.10**: Введение явных дифтонгов IY, UW, ÜW в UTL-O для маркировки родных слов.
* **v1.11 – v1.15**: Группировка 16 языков, введение расширения UTL-Extended и алгоритма автоматического определения заимствований.

---

## Основные принципы разработки

* **Фонетическая полнота**: Охват огузских, кыпчакских и карлукских ветвей.
* **AI-оптимальность**: Использование символов с высоким весом в обучающих выборках нейросетей для снижения фрагментации токенов.
* **Единообразие**: Строгое соответствие «один звук — одна буква» во всех 16 языках.
* **ASCII-совместимость**: Наличие стандартных правил записи для систем, не поддерживающих Unicode.
* **Двухуровневость**: Разделение на стандарт письма (UTL-O) и аналитический уровень для AI (UTL-P).

## Алфавит UTL (Базовый набор)

### Гласные (9 фонем)

* **Aa** — Задний ряд, неогубленный. Пример: at (лошадь). ASCII: Aa.
* **Ee** — Передний ряд, неогубленный. Пример: el (рука). ASCII: Ee.
* **Ii** — Передний ряд, неогубленный. Пример: ip (веревка). ASCII: Ii.
* **Її** — Задний ряд, неогубленный. Соответствует «ы». Пример: qiz (девушка). ASCII: I'i'.
* **Oo** — Задний ряд, огубленный. Пример: on (десять). ASCII: Oo.
* **Uu** — Задний ряд, огубленный. Пример: un (мука). ASCII: Uu.
* **Öö** — Передний ряд, огубленный. Пример: öz (сам). ASCII: Oe.
* **Üü** — Передний ряд, огубленный. Пример: gün (день). ASCII: Ue.
* **Ää** — Передний ряд, открытый неогубленный. Пример: äl (рука). ASCII: Ae.

### Основные согласные и полугласные

* **Yy** — Палатальный полугласный /j/ (Й).
* **Ww** — Лабиальный полугласный /w/. Используется в дифтонгах и начале слов.
* **Jj** — Звонкая аффриката /dʒ/ (ДЖ).
* **Čč** — Глухая аффриката /tʃ/ (Ч).
* **Šš** — Глухой фрикатив /ʃ/ (Ш).
* **Žž** — Звонкий фрикатив /ʒ/ (Ж).
* **Ğğ** — Звонкий велярный фрикатив.
* **Ŋŋ** — Велярный носовой согласный (Ң).
* **Cc** — Глухая зубная аффриката /ts/ (Ц) для заимствований.

## Правила AI-оптимизации

### Система маркировки родных слов (Дифтонги)
Для повышения точности распознавания тюркской лексики используются явные буквосочетания вместо одиночных букв в определенных позициях:
* **IY** (вместо И): biyik (высокий).
* **UW** (вместо У): suw (вода).
* **ÜW** (вместо Ү): küwn (день).
* **AW / OW**: taw (гора).

### Автоматическая обработка заимствований
Алгоритм UTL позволяет нейросети определять тип слова без дополнительных символов:
1. Если в слове нарушена гармония гласных (смешение рядов), оно считается заимствованным.
2. Склонение и суффиксация всегда ориентируются на последнюю гласную корня (например: institut -> instituttar).

## Охват языков

* **Огузская группа**: Турецкий, Азербайджанский, Туркменский, Гагаузский.
* **Кыпчакская группа**: Казахский, Кыргызский, Татарский, Башкирский, Каракалпакский, Ногайский, Кумыкский, Карачаево-балкарский, Крымскотатарский.
* **Карлукская группа**: Узбекский, Уйгурский.
* **UTL-Extended**: Якутский, Тувинский, Хакасский, Алтайский, Чувашский.
