UTL — Unified Turkic Latin (Единая Тюркская Латиница)
Версия: 1.15
Назначение: Оптимизированный стандарт письменности для 16 тюркских языков, разработанный специально для эффективной работы AI-систем и LLM-моделей.
Основные принципы проекта
 * AI-оптимальность: минимизация редкой диакритики и использование символов, которые лучше всего распознаются современными токенизаторами.
 * Единообразие: один звук всегда соответствует одной букве во всех поддерживаемых языках.
 * Двухуровневая архитектура: UTL-O для повседневного использования (фонемная запись) и UTL-P для автоматической генерации аллофонических деталей нейросетями.
 * ASCII-совместимость: каждая буква имеет однозначный эквивалент для работы в legacy-системах и коде.
 * Текстовая самодостаточность: текст на UTL не требует языковых тегов для корректного машинного распознавания.
Поддерживаемые языки
Проект охватывает три основные ветви тюркских языков (базовый уровень UTL):
 * Огузские: турецкий, азербайджанский, туркменский, гагаузский.
 * Кыпчакские: казахский, кыргызский, каракалпакский, ногайский, карачаево-балкарский, кумыкский, татарский, башкирский, крымскотатарский.
 * Карлукские: узбекский, уйгурский.
Также предусмотрен уровень UTL-Extended для сибирских (якутский, тувинский и др.) и булгарских (чувашский) языков.
Алфавит UTL (Базовый набор)
Гласные (9 букв)
 * Aa — задний ряд, неогубленный (at — лошадь).
 * Ee — передний ряд, неогубленный (el — рука).
 * Ii — передний ряд, неогубленный (ip — веревка).
 * Її — (буква i с диерезисом) — задний ряд, неогубленный (qiz — девушка). В ASCII-режиме записывается как I'.
 * Oo — задний ряд, огубленный (ot — трава).
 * Öö — передний ряд, огубленный (öz — сам).
 * Uu — задний ряд, огубленный (un — мука).
 * Üü — передний ряд, огубленный (gün — день).
 * Ää — передний ряд, открытый (äl — рука в аз., äkin — посев). Заменяет символ «Ә» для лучшей токенизации в AI.
Согласные и полугласные (выборочно)
 * Yy — палатальный полугласный (Й). Всегда используется как в слове «yol».
 * Ww — лабиальный полугласный. Используется в начале слов (watan) и в дифтонгах.
 * Jj — звонкая шипящая аффриката (ДЖ), как в английском «judge».
 * Čč — глухая шипящая аффриката (Ч).
 * Šš — глухой шипящий фрикатив (Ш).
 * Žž — звонкий шипящий фрикатив (Ж), как в слове «measure».
 * Ğğ — звонкий велярный фрикатив (глухое Г).
 * Ŋŋ — велярный носовой согласный (Ң).
 * Cc — глухая зубная аффриката (Ц). Используется преимущественно в заимствованиях (centr, policija).
Ключевые правила AI-оптимизации
Система дифтонгов
Для упрощения токенизации и устранения лишних символов (таких как И/У в кириллице), UTL использует явную запись дифтонгов в родных словах:
 * IY — заменяет «И» в родных словах (biyik — высокий).
 * UW — заменяет «У» в родных словах (suw — вода).
 * ÜW — переднерядный вариант (küwn — день в каз.).
 * AW / OW — нисходящие дифтонги (taw — гора).
Правило заимствований
В UTL-O заимствования пишутся без специальных маркеров. Статус слова и правила суффиксации определяются автоматически по последней гласной:
 * Пример: institut — последняя гласная «U» (задний ряд), поэтому суффикс будет -tar (instituttar).
 * Пример: universitet — последняя гласная «E» (передний ряд), поэтому суффикс -ter (universitetter).
ASCII-режим для разработчиков
Для систем, не поддерживающих Unicode, предусмотрен стандарт конвертации:
 * Ö → Oe, Ü → Ue, Ä → Ae.
 * Š → Sh, Č → Ch, Ž → Zh.
 * Ї (заднерядная ы) → I' (I с апострофом).
